What is HTML?

HTML or Hyper Text Markup Language is a programming language used to create webpages, browsers. The webpages can be of different categories such as- personal, from the goverment department, as a company website. Today we are using Google, Mozilla Firefox, Microsoft Edge, Safari, Opera and many more browsers and they all are also made by using html language. In matter of support it is supported by all the browsers.Now about history of html- Tim Berners-Lee, a physicist at the CERN research institute in Switzerland invented HTML in 1991. This first version consisted of 18 HTML tages. Now, there are currently about 140 HTML tags, although not all of them are supported by modern browsers.1991- Tim Berners-Lee invents HTML 1.0. If we look after the versions of html- 1993- HTML 1.0 is released. Not many developers are creating websites at this time. 1995- HTML 2.0 is published. This contains the features of HTML 1.0 plus new features. This remained the standard markup language for designing and creating websites until 1997. 1997- HTML 3.0 was invented. Here, Dave Raggett introduced a fresh draft on HTML, which improved new features of HTML and gave more powerful characteristics for webmasters in designing websites. Unfortunately, the powerful features slowed down the browser in applying further improvements. 1999- The widely-used HTML 4.0 comes out. It is very successful. 2014- HTML 5.0 is released and used worldwide. It is said to be the extended version of HTML 4.01 which was published in 2012. Some more information says that-HTML 1.0 The basic version of HTML has support for basic elements like text controls and images. HTML 2 HTML version 2.0 was developed in 1995 with basic intention of improving HTML version 1.0 Now a standard got started to develop so as to maintain common HTML 3.2 It was developed in 1997. After HTML 2.0 was developed, the next version of HTML was 3.2 With version 3.2 of HTML, HTML tags were further improved. HTML 4.01 It was developed in 1999. It extended the support of cascading styling sheets. In version 3.2, CSS were embedded in HTML page itself. HTML5 This is the latest version of HTML. For a developer, it could be used in 2014. It came up with lots of HTML tags support. Now, what are HTML tags?-HTML works same as our body works. The basic tag of HTML are <!DOCTYPE html>,<html>,<head>,<title>,<body>,etc. Description of them is as follows- Basic HTML Tags

<!DOCTYPE html>

This indicates the program that the this is a document-type website

<html>

This is the main tag that forms the website

<head>

This tag contains basic information of the website such as- title, etc.

<title>

This tag sets title of the webpage. For example- in google the tab has the name new tab.

<body>

This tag forms the body of website. Every designing part is being done with the content of the body tag.


All Tags- *

<!content>

This indicates that the content is a comment

<a>

This tag is responsible for connecting Hyper Links or multiple webpages to form a website

<abbr>

This indicates an abbreviation.

<area>

This indicates area inside a map image

<article>

This indicates an article

<audio>

This indicates an embedded sound content

<b>

This makes the text bold

<u>

This underlines the text

<div>

This makes sections by dividing the content into different parts

<img>

This tag is used to insert images. It has two parts- src, alt. src- it is used to define which image has to be inserted. alt- it is used to give the message to the user in case if the image not displayed due to connection error.

<p>

This is used to write a paragraph

<h1,h2,h3...,h6>

These are the tags that are used for writing text of different sizes. In p tag there is a default font-size, but in h1,h2,h3..,h6 there is a custom font size. h1 has maximum while h6 has minimum. The font size increases according to the numbers like 1,2,3,4,5,6.
